개요

  • 액션 중심 다크 소울류 3D 팀 프로젝트 (팀원 6명 참여)
  • 맵툴 / 네비게이션 / 연출 / 상호작용 / 인스턴싱 / 아이템 시스템 담당
  • C++ / DirectX11 / HLSL / ImGui / 자체 엔진 활용

개발 환경

  • 언어: C++
  • 도구: Visual Studio, DirectX11, HLSL, ImGui, Fmod

시연 영상 및 링크

🎥 Thymesia 시연 영상 / 🛠️ 맵툴 시연 영상 / 💻 GitHub / 📄 기술소개서 PDF

주요 구현 기능

1. Navigation 시스템

  • 픽셀 피킹 기반 셀 생성 및 월드 좌표 변환
  • 정점 시계방향 교정, 인접 관계/경계선 판정
  • Navigation Cell 내부 이동 제한, 층간 이동(y값 보정)

2. Geometry Shader 연출

  • Idle 상태 포자 꿈틀거림 (사인파 진동)
  • 피격 시 파편화 (난수 + 로드리게스 회전)
  • 돌 파편 원형 퍼짐 + 중력 낙하

3. Item 시스템

  • 아이템 등록/드롭/획득/복원 관리
  • 베지어 곡선 기반 아이템 드롭 궤적
  • UV 흔들림 + 팽창/수축 애니메이션

4. 상호작용 시스템

  • 림라이트 셰이더로 상호작용 오브젝트 강조
  • Lock-On 시스템 (UV 흐름, 연결 라인)
  • 상호작용 UI 버튼 (HUD 위치 변환)

5. 메쉬 인스턴싱 + 컬링 최적화

  • 인스턴싱 전용 오브젝트 클래스 설계
  • DrawIndexedInstanced() 기반 GPU 렌더링 최적화
  • CPU 프러스텀 컬링 (AABB vs 카메라 절두체 검사)

최적화 사례

  • 네비게이션 메쉬를 직접 생성하여 플레이어/몬스터가 벽·지형과 불필요한 충돌 검사를 하지 않도록 최적화
  • 메쉬 인스턴싱으로 수백 개의 동일 지형 드로우콜을 줄여 GPU 렌더링 효율 향상
  • 프러스텀 컬링으로 카메라 밖 오브젝트를 제거하여 CPU/GPU 불필요한 연산 감소
